Experimental Setup

  • Data: Xeno-canto API — 321 recordings (183 fox, 138 non-fox)
  • Segmentation: 3 s clips, 0.5 s overlap → 17,639 clips
  • Split: 70 / 15 / 15 stratified (train / val / test)
  • Features: MFCC 40-coeff → 80-d vector; log-mel spectrograms 128 bands
  • Hardware: Apple M-series (MPS) / CPU fallback

Model Justification

  • SVM (RBF) — strong margin-based baseline for MFCC features
  • Random Forest — ensemble baseline, handles non-linear boundaries
  • Gradient Boosting — sequential ensemble for comparison
  • EfficientNet-B0 — ImageNet-pretrained CNN on spectrograms
  • CNN: AdamW, CosineAnnealingLR, weighted CE, SpecAugment

Reproducibility

How to Reproduce

  • Clone repo & run pip install -r requirements.txt
  • Open Final.ipynb — single notebook runs entire pipeline
  • Downloads data, segments, extracts features, trains, evaluates
  • Random seed 42 for all splits & model init
  • Auto-detects MPS / CUDA / CPU

Tech Stack

  • Python · librosa · soundfile
  • PyTorch · torchvision (EfficientNet-B0)
  • scikit-learn (SVM, RF, GB)
  • Gradio (interactive demo)
  • Chart.js (this dashboard)
